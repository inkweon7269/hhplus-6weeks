# 잔액 충전/조회 
## 정책
- 1일 최대 충전 금액은 무제한합니다.
- 충전된 금액은 기간과 무관하게 소멸되지 않습니다.
- 결제 시에는 충전 금액에서 차감됩니다.
- 잔액은 보인만 조회 가능합니다.

## 시퀀스 다이어그램
```mermaid
sequenceDiagram
    participant 사용자
    participant 서버

    사용자->>서버: 잔액 충전 요청(사용자ID, 금액)
    서버->>서버: 잔액 충전 처리
    서버-->>사용자: 충전 결과 반환

    사용자->>서버: 잔액 조회 요청(사용자ID)
    서버->>서버: 잔액 정보 조회
    서버-->>사용자: 잔액 정보 반환
```
